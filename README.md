# Titre du projet

AVEC-Spot

Projet sur l'API de Spotify

## Installation 

npm i
create a .env with two global variables 

REACT_APP_CLIENT_ID= your client id from Spotify API
REACT_APP_CLIENT_SECRET= your client secret from Spotify API
